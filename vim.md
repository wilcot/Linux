# vim

## Commands
| Command | Action |
| ------- | ------ |
| `$` | move to the end of the line
| `A` | move to the end of the line and begin editing
| `dd` | delete a line
| `3dd` | delete three lines (substitute number for desired amount of line)
| `yy` | copy a line
| `p` | past a line
| `:u` | undo
| `Ctrl+r` | redo
| `wq` | write then quit
| `q!` | quit and don't save
